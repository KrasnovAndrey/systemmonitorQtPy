# System Monitor

## Автор

Краснов Андрей Юрьевич, 9 класс

## Описание идеи

System Monitor - это кроссплатформенное приложение для мониторинга компьютерной системы. Программа позволяет отслеживать состояние аппаратных компонентов, системных служб и процессов. Основная цель проекта - создать удобный инструмент для диагностики и контроля работоспособности системы.

## Описание реализации

Приложение построено на архитектуре с разделением на кастомные UI-компоненты и функциональные модули. Основные классы включают CustomWindow для главного окна, CustomSidebar для навигации, CustomPanel и CustomCard для отображения информации.

Также реализована система шаблонов служб в формате CSV, которая позволяет сравнивать текущее состояние системы с эталонным и выявлять потенциальные проблемы. Все компоненты интерфейса выполнены в единой цветовой схеме с использованием CSS стилей PyQt6.

## Технологии и библиотеки

**Основные технологии:**

- Python 3.8+
- PyQt6 - графический интерфейс
- psutil - системная информация
- subprocess - выполнение системных команд
- csv - работа с шаблонами служб

**Необходимые для запуска библиотеки:**

```
pip install PyQt6 psutil
```
Или
```
python -m venv venv
```
Для Windows используйте:
```
venv\Scripts\activate
```
Для macOS и Linux используйте:
```
source venv/bin/activate
```
```
pip install PyQt6 psutil
```
**Поддерживаемые операционные системы:**

- Linux Arch-подобные
- Windows 10/11
